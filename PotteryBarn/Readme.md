# PotteryBarn

  * Public build extension that addes prefabs to the hammers build options

## Instructions

  1. Extract `PotteryBarn.dll` to your `/Valheim/BepInEx/plugins/` folder.
  2. Addition Prefabs should be in the Hammers build options.
